## Deskripsi Project
MyNotes adalah aplikasi catatan harian berbasis Android
yang dirancang dengan pendekatan UI/UX untuk memudahkan
pengguna mencatat dan mengelola catatan harian.

## Fitur
- Tambah catatan
- Lihat detail catatan
- Edit & hapus catatan
- Halaman Settings
